In order to view this site, please clone the repository to your local machine.

Open the file containing the project in VSCode.

Open a terminal and type:
npm install 
this will install all dependencies

In the terminal, then type: 
gulp
this will open the portfolio on localhost port 3000 on your machine.

Notes on portfolio section:
Hover over each image, the + symbol will enlarge the picture
The link symbol will link to the github repository for each project.
